
## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy on Vercel

This project is ready for Vercel deployment with `vercel.json` configured.

1. Install the Vercel CLI if needed:
   `npm install -g vercel`
2. Deploy:
   `vercel` or `npx vercel`

Alternatively, connect the repository in the Vercel dashboard and use:
- Build command: `npm run build`
- Output directory: `dist`
